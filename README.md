# 🏥 心血管专科知识图谱 Web 可视化

> 基于 Neo4j 的心血管内科专病知识图谱，支持 76 种心血管疾病、17 个知识维度的交互式浏览与分析。

## 功能模块

| 模块 | 说明 |
|------|------|
| 📊 数据总览 | 专科知识路径、疾病大类完整率、17 维度成熟度、质量缺口 |
| 🧭 图谱探索 | 按疾病查看 17 维度知识、关系网络、实体详情 |
| 🕸️ 网络探索 | 交互式知识图谱，支持维度筛选、路径查找、全屏浏览 |
| 🗺️ 数据覆盖分析 | 76 种专病 × 17 维度矩阵，定位缺失与薄弱维度 |
| 🔍 临床诊断模拟 | 输入病例信息，匹配候选疾病 |
| 📐 图谱数据字典 | 实体类型、关系类型、疾病分类统计 |
| 📘 Schema 标准 | 专科专病图谱建模标准、字段约束、质量规则 |
| 🧬 医学术语知识库 | 按维度浏览所有医学术语，支持搜索和关联查看 |
| ⚙️ 系统配置 | 维护 Neo4j 图谱服务地址和连接参数 |

## 技术栈

- **前端**: HTML + CSS + ECharts 5.x（单文件 SPA）
- **后端**: Python Flask → Neo4j Cypher
- **数据库**: Neo4j 图数据库

## 快速启动

```bash
# 启动后端
cd server
python main.py  # 默认 0.0.0.0:4000

# 前端（由 Flask 静态托管）
http://192.168.3.27:4001/
```

## 数据规模

- 76 种心血管专病
- 17 个知识维度（症状、体征、检查、检验、药物、手术、治疗方案……）
- 26,249 个知识节点
- 9,555 条关系边
